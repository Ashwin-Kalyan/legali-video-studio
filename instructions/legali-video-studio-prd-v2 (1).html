01 — Executive Overview

What We're Building & Why Now

Legali AI runs four brand voices across Instagram, TikTok, and LinkedIn. Today, video production is manual, analytics are siloed across native dashboards, and brand consistency depends entirely on Ira's personal oversight. This does not scale past a 3-person team.

Legali Video Studio is an internal AI-powered platform with three sequentially unlocking modules. Module 0 must ship first — nothing else works without brand kits trained.

🎓
Module 0 — Brand Onboarding

Conversational 7-question brand training flow per brand. Outputs a structured brand kit (voice, palette, tone, prohibited phrases, CTA, hashtags) that all downstream modules consume. Prerequisite for everything.

📊
Module 1 — Video Analytics

Unified cross-platform dashboard (Instagram + TikTok + LinkedIn) with AI weekly insight digest, brand health scorecards, content-type attribution, and waitlist conversion tracking.

✂️
Module 2 — AI Video Cut Studio

Upload raw footage → brand kit auto-applies → AI generates 3 candidate cuts → voice record or clone → export in 9:16/1:1/4:5 → direct publish or download. Target: <2 min to first cut.

3.3 — Post-Onboarding: Visual Identity Configuration

After the 7 conversational questions, the user is taken to a structured form to complete the brand kit's visual layer:

  • Primary & secondary color pickers (hex) — pre-populated with Legali brand defaults per brand
  • Font family selector with live preview (Google Fonts + system fonts)
  • Logo upload (PNG/SVG, max 2MB) with placement preview: top-left, bottom-left, bottom-right
  • Transition style selector: Soft Fade / Clean Cut / Snap / Bounce — with animated preview
  • Caption style selector: word-by-word / line-by-line / highlight key terms — with preview
  • Trauma-informed mode toggle (default ON for Lea, OFF for all others)
  • Music mood selector: Soft Emotional / Confident / Playful / Corporate / None
  • CTA template text field: e.g. "Join the waitlist at lea.legali.ai"
  • Prohibited phrases: tag input — these are injected into every AI generation prompt as hard exclusions
  • Hashtag library: tag input — auto-appended to exported captions per platform limits

5.2 — Core User Flow

Step 1
Upload Raw Footage

Browser drag-and-drop. Accepts MP4, MOV, HEVC up to 4GB total. Chunked upload to Supabase Storage. Auto-transcription (Whisper v3) begins immediately. Waveform + transcript preview shown as it streams in.

Step 2
Select Brand Kit + Format Template

Dropdown of trained brand kits. Format picker with animated previews (Awareness Hook, Founder Story, UGC Review, Listicle, Q&A, Product Demo, Day in My Life). Brand kit immediately previews color/font choices.

Step 3
AI Context Analysis

4-layer analysis runs in parallel: speech scoring, visual quality scoring, brand voice alignment, format constraint checking. Produces 3 ranked candidate edits with engagement scores. Time target: <30s from transcript ready.

Step 4
Review, Fine-tune & Voice

Split-screen: preview left, controls right. Drag-and-drop timeline for reordering. Per-clip trim handles. In-browser mic recording with noise removal and loudness normalization. Inline caption editing. Regenerate with preference nudge.

Step 5
Export & Publish

One-click resize for 9:16 / 1:1 / 4:5 / 16:9. Export as MP4 + optional SRT. Direct push to Meta Business Suite or TikTok for Business with caption pre-filled. Schedule future publish via calendar picker.

5.3 — AI Context Awareness Engine

🎤
Speech Layer

Whisper v3 transcript → segment classification. Each 5s segment scored on: clarity (0–1), emotional intensity (0–1), information density (0–1), hook potential (0–1). Top 20% of segments form the candidate pool.

👁️
Visual Layer

Frame sampling at 1fps via GPT-4V. Scores per frame: eye contact (0–1), lighting quality (0–1), background clutter (0–1), motion stability (0–1). Frames below 0.5 composite are excluded.

🧠
Brand Voice Layer

Brand kit voice description + prohibited phrases injected as system prompt context. Claude evaluates each candidate segment for voice alignment score (0–1). Clips below 0.6 are deprioritized.

📐
Format Constraint Layer

Each format template defines: target duration ± 5s, pacing style (fast/slow/dramatic), CTA placement (final 5s), hook window (first 3s must be >0.8 hook score). AI selects clips to satisfy all constraints.

06 — Prompt Engineering Specs NEW

AI Prompt Architecture

All Claude API calls follow a consistent 3-layer prompt structure: System context (brand kit + safety rules), Task instructions (what to generate), Input data (transcript segments / analytics). Below are the production prompt specs for each AI call.

6.1 — Cut Generation Prompt

// SYSTEM PROMPT — injected once per project
SYSTEM:
You are an expert short-form video editor working for {brand_kit.company_name}.
// Brand voice context
Brand voice: {brand_kit.voice_description}
Tone tags: {brand_kit.tone_tags.join(', ')}
Target audience: {brand_kit.target_audience}
Core product: {brand_kit.product_description}
// Hard constraints
NEVER include the following phrases in any generated text: {brand_kit.prohibited_phrases.join(', ')}
if (brand_kit.trauma_informed_mode) {
  Apply trauma-informed language standards. Avoid re-traumatizing phrasing,
  minimizing language, or content that centers perpetrators over survivors.
  Flag any segment transcript that contains: [victim-blaming phrases list]
}

USER:
Format template: {template.name}
Target duration: {template.duration_min}–{template.duration_max}s
Pacing: {template.pacing_style}

Transcript segments with scores:
[{segment_id, start_ms, end_ms, text, speech_score, visual_score}]

Generate 3 candidate edits. For each, return:
- Selected segment IDs in order
- Hook text (first 3 seconds, max 8 words)
- Caption narration script (brand voice aligned)
- CTA text (last 5 seconds)
- Predicted engagement score (0–10) with breakdown
Respond in JSON only.

6.2 — Weekly Analytics Insight Prompt

SYSTEM:
You are the analytics advisor for Legali AI, a legal technology company.
Write in a direct, data-driven, actionable tone for the CEO.
Focus on: what drove performance, what underperformed and why, and 1 specific recommended action.
Reference specific posts by name/date. Be concrete — no generic advice.

USER:
Analytics data for the past 7 days across all brands:
{analytics_summary_json}

Generate:
1. A 2-sentence "week in review" summary
2. Top 3 performance observations (with post references)
3. 1 pattern identified (hook language, posting time, content type)
4. 1 specific recommended action for next week
5. 1 metric to watch
Max 250 words. Return as structured JSON with keys: summary, observations[], pattern, action, watch_metric.

6.3 — Trauma-Informed Content Safety Check

// Runs on ALL Lea brand AI-generated text before display
SYSTEM:
You are a trauma-informed content reviewer for Lea by Legali,
an AI companion for domestic violence survivors.
Review the following generated text for:
- Victim-blaming language (e.g. "why didn't she leave", "she provoked")
- Minimizing language (e.g. "just abuse", "it wasn't that bad")
- Perpetrator-centering (focusing on abuser's actions/emotions over survivor agency)
- Crisis triggers without appropriate warning (graphic descriptions of violence)
- Anything that could discourage a survivor from seeking help

USER:
Review this generated caption/script:
"{generated_text}"

Return JSON:
{ passed: boolean, flags: [{phrase, reason, suggested_replacement}], revised_text: string }

  differentiators    TEXT[],
  voice_description  TEXT,
  tone_tags          TEXT[],

  font_family        TEXT,
  logo_url           TEXT,
  logo_position      TEXT CHECK (logo_position IN ('top-left','bottom-left','bottom-right')),
  transition_style   TEXT CHECK (transition_style IN ('soft-fade','clean-cut','snap','bounce')),
  caption_style      TEXT CHECK (caption_style IN ('word-by-word','line-by-line','highlight')),
  music_mood         TEXT,
  cta_template       TEXT,
